This is a source/doc acceptance demo for the initial RTA repository bootstrap.

1. Open [mjweavermount/rta](https://github.com/mjweavermount/rta).
2. Confirm the repo owner/name are correct.

4. Confirm the local path exists:
   - `/Users/virgil/Developer/rta`

The repo is present, named correctly, and contains the seed docs/package skeleton needed for the next implementation card.
